
# MapBook (IOS Swift)

Seçtiğiniz konuma başlık ve not ekleyerek haritalardan güncel konumunuzdan seçilen konuma navigasyon gösteren haritalı not tutma uygulaması.


## Kullanılan Teknolojiler

- Swift

- Storyboard

- CoreData

- MapKit
